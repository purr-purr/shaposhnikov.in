## ⚙️ Features

- Webpack 5
- SCSS
- JavaScript
- JQuery
- Autoprefixer
- Image Minify
- Dev Server


## 🏁 Quick start project

# Install dependencies:

npm install

# Server with live reload at http://localhost:8080/

npm run dev

## 🏁 Production build

# Output will be at dist/ folder

npm run build
