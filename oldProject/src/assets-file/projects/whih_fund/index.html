Про Фонд

Як працює Фонд

Фонд заснований в 2016 році. З початку повномасштабного вторгнення, наш фонд почав діяльність задля допомоги усім, хто страждає від суворої агресії ворога. Нами було створено усі можливі варіанти оплат для донаторів, задля швидкого перерахування коштів. За кожен перерахований донат для населення України, ми будемо звітувати у даному розділі.
